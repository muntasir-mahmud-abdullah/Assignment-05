Frequently asked important questions

1. What is the Document Object Model (DOM)?

The Document Object Model (DOM) is a programming interface for web documents. It represents the structure of a webpage as a tree of objects where each object corresponds to a part of document, like an element attribute or text. The DOM allow javascript to access,modify and manipulate and structure and content of the webpage dynamically.The DOM also handles events,allowing javascript to respond to user actions.

2. How do you select an element from the DOM?

1.Select a single element by its unique id. 2.Select all elements with specific class name and returns HTMLCollection (array like object). 3.Select all elements with specific tag name and returns HTMLCollection (array like object). 4.Select the first element that matches the css selector(class,id, tag etc.). 5.Select all elements that match the css selector(class,id, tag etc.) and returns Nodelist(like array)

3. What is event delegation in the context of the DOM, and why is it useful?

Event delegation in the DOM is a technique where instead of attaching of event listeners for multiple child elements you attach a single event listeners for parent element. This event listener can then handle events that 'bubble up' from the child elements.It is useful for performance due to consuming less memory, for dynamically update contents and for simplify code.

4. How do you manipulate an element's attributes and styles using the DOM?

I manipulate an element's attributes using the DOM through getAttribute(),removeAttribute(),addAttribute(), attribute.property-name and I manipulate an element's style using the DOM through element.style.property-name and also using element.classlist.remove('class-name'), element.classlist.add('class-name') and element.classlist.toggle('class-name').
